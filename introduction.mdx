## Welcome to Mangopay PHP SDK

The Mangopay PHP SDK is a client library that simplifies integration with the [Mangopay REST API](https://docs.mangopay.com/endpoints/v2.01). It provides a comprehensive set of tools for managing users, wallets, payments, transfers, and more.

## What is Mangopay?

Mangopay is a payment infrastructure platform that enables marketplaces and platforms to accept payments, manage user accounts, and facilitate transfers between users. The PHP SDK abstracts the complexity of direct API calls and provides an intuitive object-oriented interface.

## Requirements

To use the Mangopay PHP SDK, you need:

- **PHP 5.6 or newer**
- **cURL** (included and enabled in standard PHP distributions)
- **OpenSSL** (included and enabled in standard PHP distributions)
- **psr/log** v1.0 (automatically handled by Composer)

<Note>
  While Composer is not mandatory, it is **strongly recommended** for managing dependencies, particularly the PSR Log library.
</Note>

## API Version Compatibility

<Info>
  Since v2.1 of this SDK, you must use at least v2.01 of the Mangopay API. See the [API v2.01 overview](https://docs.mangopay.com/api-v2-01-overview/) for migration details.
</Info>

## Getting Your API Credentials

Before you begin, you'll need Mangopay API credentials:

<Steps>
  <Step title="Choose Your Environment">
    - **Sandbox**: Free testing environment for development
    - **Production**: Live environment for real transactions
  </Step>
  <Step title="Register Your Account">
    Visit the [Mangopay registration page](https://www.mangopay.com/start/) to create an account.
    
    <Warning>
      Production account validation involves several steps. Start the process well before your intended go-live date.
    </Warning>
  </Step>
  <Step title="Retrieve Your Credentials">
    After registration, you'll receive:
    - **Client ID**: Your unique identifier
    - **API Key**: Your authentication password (also called Client Password)
  </Step>
</Steps>
